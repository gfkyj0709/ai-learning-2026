# ai-learning-2026
AI/LLM 학습 기록 (2026)


---

### Phase 1 (3~5월): 기초 환경 + 개념
```
✅ 완료
- Proxmox Ubuntu VM 구축
- Ollama 설치 및 API 호출
- LLM 핵심 개념 정리
- GitHub 세팅

🔜 진행중
- Spring Boot + Ollama 연동
- Spring Cloud 기초
- Prompt Engineering 실습
- 영어 루틴 정착
```

---

### Phase 2 (6~8월): RAG + MSA + 자격증
```
- RAG 구현 (LangChain4j + ChromaDB)
- Embedding + Semantic Search 실습
- Spring Cloud MSA 실습
  - Gateway / Eureka / Config / OpenFeign / Circuit Breaker
- AI 서비스 MVP 개발 및 배포
- 클라우드 자격증 취득 (AWS or GCP)
```

---

### Phase 3 (9~11월): Agent + 고도화 + 포트폴리오
```
- AI Agent 구현 (LangGraph)
- Function Calling (기존 시스템 + AI 연동)
- Multimodal 실습
  - Whisper (STT) 연동
  - 음성 입력 서비스 구현
- Spring Cloud + AI 서비스 연계
- 포트폴리오 완성
- 영어 기술 콘텐츠 제작 시작
```

---

### Phase 4 (12~2월): 고급 기술 + 2번째 서비스
```
- Fine-tuning 맛보기 (Google Colab)
- LLM Evaluation (품질 관리)
- 두 번째 서비스 개발
- 두 번째 자격증 준비
- 1년 회고 + 유튜브 콘텐츠 정리
```

---

## 최종 목표 스택 (2027년 초)

```
Backend   : Java + Spring Boot + Spring Cloud MSA
AI        : RAG + Agent + Function Calling + Whisper
Infra     : AWS or GCP + Docker
DB        : PostgreSQL + ChromaDB (벡터DB)
자격증    : AWS SAA or GCP ACE + 하나 더
포트폴리오: AI 서비스 2개 + GitHub + 유튜브
```

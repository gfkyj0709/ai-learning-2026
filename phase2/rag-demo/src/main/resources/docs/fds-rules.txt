FDS 탐지 규칙 및 시나리오

1. 시간대 이상 거래 규칙
- 새벽 00시~04시 사이 50만원 이상 이체 발생 시 추가 인증 요구
- 고객 평소 거래 시간대 외 대규모 거래 시 알림 발송
- 해외 IP에서 접속 후 국내 거래 발생 시 의심 거래로 분류

2. 금액 이상 탐지 규칙
- 일일 한도의 80% 이상을 단시간(1시간) 내 사용 시 차단
- 동일 계좌로 24시간 내 반복 소액 이체(쪼개기) 탐지
- 평소 거래 금액 대비 300% 초과 거래 발생 시 검토 대상

3. 기기 및 접속 이상 규칙
- 최초 접속 기기에서 고액 거래 시도 시 본인 확인 절차 강화
- 단시간 내 다수 기기에서 동일 계좌 접근 차단
- VPN/Proxy 접속 탐지 후 거래 시 모니터링 강화

4. 수취인 이상 탐지 규칙
- 신규 수취인에게 첫 거래 시 고액(100만원 이상) 이체 제한
- 금융사기 피해 계좌로 등록된 계좌로의 이체 즉시 차단
- 동일 수취인에게 짧은 시간 내 반복 이체 시 확인 요청

5. 패턴 기반 탐지
- 계좌 생성 후 72시간 이내 대규모 출금은 고위험으로 분류
- 입금 직후 즉시 전액 출금 패턴 탐지 (머니뮬 의심)
- 로그인 실패 반복 후 성공한 경우 고위험 세션으로 관리
